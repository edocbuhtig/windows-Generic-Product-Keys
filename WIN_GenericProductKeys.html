    List of Generic Product Keys to Install Windows 10 Editions


    Generic keys (aka: "default keys") for Windows 10 from Microsoft will allow you to install or upgrade to a specific Windows 10 edition you want, but will not activate it.

    Using a generic key can be helpful if you wanted to install or upgrade to a specific Windows 10 edition for evaluation or testing on a PC or virtual machine, or just don't have a digital license or your genuine product key currently available and you will activate later when able.

    This tutorial will provide you with a list of RTM (retail) and KMS generic keys (default keys) for all editions of Windows 10.

    KMS client setup keys sources:
    • KMS client setup keys | Microsoft Docs (newer)
    • KMS Client Setup Keys | Microsoft Docs
